Home Automation
Using ARDUINO

Designed Wireless Home Automation control using WLAN technology and RF remote control to control selected home devices. Each room device has its own board connected to the NodeMCU through the relay circuit. The software uses assembly language for microcontroller programming and ARDUINO language for smartphone communication and GUI. The system is low cost and flexible with an increasing variety of devices.

Modes: Manual Mode, Automatic, Security Mode

Tech used: ARDUINO, NodeMCU, Ultra Sonic Sensor, Breadboards, Jumper Cable
